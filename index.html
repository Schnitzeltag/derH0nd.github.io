# Was das hier ist

Diese Seite ist ein Gegenentwurf. Gebaut mit HTML, CSS und ein bisschen JavaScript. Kein Framework, kein CMS, kein Cookie-Banner.

Inspiriert vom Web Revival – der Idee, dass das Internet wieder den Menschen gehören sollte, die es nutzen. Nicht den Plattformen, die es monetarisieren.
